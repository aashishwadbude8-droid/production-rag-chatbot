# Production RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built using:

- Streamlit
- LangChain
- FAISS
- BM25
- Groq LLM
- Sentence Transformers

## Features

- PDF Upload
- Hybrid Search (Semantic + BM25)
- Conversational Memory
- Fast Retrieval
- AI-powered Answers
